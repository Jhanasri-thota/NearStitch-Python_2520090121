# NearStitch – AI Powered Fashion Designer Booking System

An intelligent Python-based fashion designer and tailor booking system developed as a CFAI Project using Artificial Intelligence concepts from CO1 to CO6.

---

# 📌 Project Description

NearStitch is a console-based AI-powered tailoring and fashion designer booking system.  
The project helps users search nearby tailor shops, compare ratings, select dress models, book stitching appointments, and generate automated bills.

This project demonstrates the implementation of Artificial Intelligence concepts such as:

- Intelligent Agents
- Search Algorithms
- Constraint Satisfaction
- Utility-Based Decision Making
- Probabilistic Reasoning
- Performance Profiling

The system is designed as a practical AI application for real-world tailoring and fashion service management.

# 🧠 AI Concepts Implemented

## ✅ CO1 – Intelligent Agent & Knowledge Representation

### PEAS Model
- Performance → Accurate recommendations
- Environment → Users, designers, booking system
- Actuators → Booking confirmation
- Sensors → Ratings, budget, user input

### Knowledge Representation
- Dataclass representation
- Lists
- Dictionaries

---

## ✅ CO2 – Search Algorithms

The project implements the following AI search techniques:

### BFS (Breadth First Search)
Searches designers level by level.

### DFS (Depth First Search)
Uses recursion to search designers deeply.

### UCS (Uniform Cost Search)
Selects the lowest cost designer.

### A* Search
Uses cost + heuristic for intelligent recommendation.

### Greedy Best First Search
Chooses designer with highest rating.

---

## ✅ CO4 – Decision Making & Game Theory

### Utility Function
Calculates customer satisfaction score.

### Minimax Concept
Simulates customer vs designer advantage.

Demonstrates branch pruning optimization.

---

## ✅ CO5 – Probabilistic Reasoning

### Bayesian Probability
Predicts recommendation confidence.

### Sensor Fusion
Combines rating, availability, and cost.
